Требование:
**Игровая студия среднего размера (примерно 1 миллион пользователей) хочет реализовать функцию чата для своей игры.**

**Требования:**

- Игроки должны иметь возможность создавать публичные чат-комнаты, присоединяться к ним и подробно обсуждать контент с другими игроками.
    
- Можно предположить, что у каждой учетной записи игрока есть уникальный, универсально идентифицируемый `playerId`, связанный с этим человеком.
    
- Кроме того, хотя сама игра является исключительно мобильным приложением, студия хочет, чтобы функциональность чата была доступна на любом устройстве, включая веб, десктоп и (очевидно) мобильные платформы. (Их текущие игры написаны с использованием нативных мобильных инструментов — Kotlin/Android и Swift/iOS.)
    
- У студии также есть несколько новых проектов, которые почти готовы к выпуску, и она хочет иметь возможность использовать **одну и ту же** чат-систему для каждой из этих новых игр.
